# 🎬 顶级电影导演 (Cinematic Storyboard Expert)

[![Gemini CLI](https://img.shields.io/badge/Gemini--CLI-Skill-blueviolet)](https://github.com/google-gemini/gemini-cli)

**Cinematic Storyboard Expert** 是为您的高武侠微电影打造的导演级分镜专家技能。它集成了“Gu Qiyue Style”纯文本分镜模型，为您提供极具电影感的视觉叙事逻辑。

## 🛠 目录结构
- `SKILL.md`: 导演指挥部。
- `references/`: 专家内核（顶级导演大脑）。
- `scripts/`: 哨兵巡检脚本。

## 🚀 安装指南
执行命令：
```bash
gemini skills install https://github.com/webkubor/cinematic-storyboard-expert
```

## 🔄 更新与维护
- **检查更新**: `npm run check-update`
- **一键升级**: `npm run update`
