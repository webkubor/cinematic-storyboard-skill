# 🎬 剧本医生 (Cinematic Storyboard Expert)

[![Gemini CLI](https://img.shields.io/badge/Gemini--CLI-Skill-blueviolet)](https://github.com/google-gemini/gemini-cli)

**剧本医生 (Cinematic Storyboard Expert)** 是一个专注于优化电影分镜、视觉叙事逻辑以及 AI 视频模型 (Sora/Kling/Runway) 提示词的专业技能。

## 🛠 目录结构

- `SKILL.md`: 技能核心逻辑（AI 大脑）。

## 🚀 安装指南

确保您已安装 [Gemini CLI](https://github.com/google-gemini/gemini-cli)，执行：

```bash
gemini skills install .
```

## 📖 使用场景

1. **分镜优化**: 将粗糙的创意转化为具备视觉表现力的镜头拆解。
2. **AI 视频提示词**: 生成高质量的跨模型视频提示词方案。
3. **镜头语言设计**: 规划电影感的运镜、构图与光影。
